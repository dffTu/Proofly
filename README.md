# Proofly

Интерактивный граф математических теорем и аксиом с доказательствами, визуализированный как направленный граф зависимостей.

## Возможности

- **Интерактивный граф** — навигация, масштабирование, ссылки
- **Доказательства** — клик по узлу открывает панель с доказательством в LaTeX
- **Двуязычность** — переключение между русским и английским
- **Поиск** — мгновенный поиск по названиям и описаниям теорем

## Стек

Django · PostgreSQL · D3.js · KaTeX · Docker

## Быстрый старт

```bash
docker-compose up --build
```

Откройте [localhost](http://localhost) в браузере.

## Структура проекта

```
data/           — статические данные
graph/          — Django-приложение (модели, API, шаблоны, JS/CSS)
scripts/        — загрузка данных, парсинг, перевод
sql/            — миграции (golang-migrate)
tests/          — Selenium-тесты
```
